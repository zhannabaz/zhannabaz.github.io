# zhannabaz.github.io

Personal webpage.

## Local testing

This site is a static page, so you can preview it with a simple local web server.

From the project root, run:

```sh
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

Notes:

- The automatic redirect is disabled on `localhost` and `127.0.0.1` so you can inspect changes locally.
- On GitHub Pages, the redirect still works as before.
